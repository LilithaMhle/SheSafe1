# SheSafe

SheSafe is a menstrual product awareness platform designed to help female students evaluate the safety and transparency of menstrual products. The system provides ingredient awareness, safety indicators, and peer reviews to support informed purchasing decisions.

## Project Goals
- Promote menstrual product safety awareness
- Provide ingredient transparency
- Allow users to review menstrual products
- Educate users about potentially harmful chemicals

## Agile Development
